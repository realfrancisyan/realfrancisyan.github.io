Embed Medium Posts in Your Awesome Tumblr Sites

Due to some blog services stripping script tags and thus preventing Medium embeds from functioning, Tumblr is currently not available for users to embed Medium stories directly in posts using its editor. But after several tries I accidentally worked out a way that could make it functional with just a few steps.

Add Necessary Scripts to Your Theme

Tumblr provides us great benefits to edit HTML and CSS code which enable us to create complex layouts. As a result, we can intuitively add the following single line of code to our themes to meet the requirement of Medium embeds.

<script async src="https://static.medium.com/embed.js"></script>

Embed Story

You are now free to embed Medium posts to your awesome blog. Please note that it is not necessary to insert the script that has already been added to your theme. All you need to do is to copy the line of code that starts with the “a” tag, such as:

<a class="m-story" data-collapsed="true" ...></a>
