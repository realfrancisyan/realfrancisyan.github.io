1. let 和 const

两者都为块级作用域。

let 声明局部变量，而 const 声明常量（不可修改）。

2. 箭头函数

箭头函数最直观的三个特点：

不需要 function 关键字来创建函数。
省略 return 关键字。
继承当前上下文的 this 关键字。
